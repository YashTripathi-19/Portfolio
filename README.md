## ✨ Features

🎨 **Interactive 3D Graphics** - Immersive WebGL-powered 3D models  
⚡ **Lightning Fast** - Built with Vite for optimal performance  
📱 **Fully Responsive** - Seamless experience across all devices  
🎭 **Smooth Animations** - Powered by Framer Motion  
📧 **Contact Form** - Direct communication via EmailJS  
🔄 **Auto-Updated Projects** - Fetches latest repos from GitHub  

---

## 🎯 Quick Start

```bash
# Clone the repository
git clone https://github.com/YashTripathi-19/Portfolio.git
cd Portfolio

# Install dependencies
npm install --legacy-peer-deps

# Start development server
npm run dev

# Build for production
npm run build
```

Visit `http://localhost:5173` to see the portfolio in action! 🎉

## 🚀 Deployment

This portfolio is deployed on **Netlify** with automatic deployments on every push to `main`.

[![Netlify Status](https://api.netlify.com/api/v1/badges/your-badge-id/deploy-status)](https://app.netlify.com/sites/yashtripathi-19/deploys)

### Deploy Your Own

1. Fork this repository
2. Connect to [Netlify](https://app.netlify.com)
3. Add environment variables (see [Configuration Guide](./docs/guides/CONFIGURATION.md))
4. Deploy! 🎊
