# Portfolio Website - Yash Tripathi

Modern 3D portfolio website showcasing backend development projects and experience.

## 🚀 Built With

- **React.js** - Frontend framework
- **Three.js** - 3D graphics and animations
- **Framer Motion** - Smooth animations
- **Tailwind CSS** - Styling
- **EmailJS** - Contact form integration
- **Vite** - Build tool

## ✨ Features

- Interactive 3D models and animations
- Smooth scrolling and transitions
- Responsive design for all devices
- Working contact form with EmailJS
- Project showcase with live demos
- Work experience timeline
- Tech stack visualization

## 🛠️ Projects Showcased

1. **CarbonSync** - Carbon Compliance Platform (Spring Boot, REST API)
2. **Auberon Pharmaceuticals** - Pharmaceutical Management System (Node.js, TypeScript)
3. **Block-Chained To-Do List** - Decentralized DApp (Solidity, Ethereum)

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/YashTripathi-19/Portfolio.git

# Navigate to project directory
cd Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

## 🌐 Deployment

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```
